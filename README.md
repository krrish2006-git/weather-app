# 🌦️ Weather Detection App

![Python](https://img.shields.io/badge/Python-3.x-blue)

## 📌 OVERVIEW
This is a Python-based weather application that fetches real-time weather data using an API and displays important details like temperature, humidity, and weather conditions.

## 🚀 FEATURES
- 🌍 Get real-time weather data
- 🔍 Search weather by city
- 🌡️ Displays temperature, humidity, and conditions
- ⚠️ Handles invalid inputs

 🛠️ Tech Stack
- Python
- Requests Library
- Weather API

## ▶️ HOW TO RUN

1. Clone the repository:
   git clone https://github.com/krrish2006-git/weather-app.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the program:
   python weather.py
